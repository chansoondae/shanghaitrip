## 기술 스택

- **Framework**: Next.js 16 (App Router) · React 19 · TypeScript
- **Styling**: Tailwind CSS v4 (모바일 퍼스트)
- **Icons**: lucide-react, framer-motion

## 주요 기능

- 일자별 타임라인 상세 일정
- 각 스팟 구글맵 바로가기 버튼
- 전시 및 식당 상세 정보
- 글자 크기 조절 (접근성 고려)
- 하단 탭 네비게이션
- 인솔자·가이드 전화 바로걸기 버튼

## 로컬 실행

```bash
npm install
npm run dev      # 개발 모드 (이미지·코드 변경 즉시 반영)
npm run build    # 프로덕션 빌드
npm run start    # 프로덕션 서버 실행
```

[http://localhost:3000](http://localhost:3000) 접속
